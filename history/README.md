# Historial de Sesiones

Cada archivo `YYYY-MM-DD_HHMMSS_<session-id>.md` registra una sesión de trabajo.

## Formato

```markdown
# Sesión: <session-id>
- Fecha: YYYY-MM-DD HH:MM:SS
- Objetivo: ...

## Pasos realizados
1. ...
2. ...

## Archivos modificados
- `ruta/archivo.py` — descripción

## Decisiones clave
- ...

## Estado actual
- ...
```

## Para Agentes

Al iniciar una nueva sesión, leer el archivo más reciente de `history/`
para entender el contexto actual del proyecto.
